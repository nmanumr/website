## Development

To work on this page: 

clone the repo

```bash
$ git clone https://github.com/directus/website
```

install the dependencies

```bash
$ npm install
```

build the front-end

```bash
$ npm run dev
```

## Deployment
```
npm run build
cd dist
git init
git add -A
git commit -m 'deploy'
git push -f https://github.com/directus/website master:gh-pages
```
