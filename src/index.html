Directus provides a dynamic API for any new or existing SQL database. Choose between REST or GraphQL endpoints and you'll be knee-deep in JSON before you know it!

If your project application is on the same server as its database, you can remove that last shred of latency and connect directly with SQL queries. Remember, you are in control of the actual database schema – Directus stays out of the way.

Directus can be used as a Standalone Application. Create anything from internal productivity tools to business intelligence platforms. No development required.

A customizable toolkit, not a one-size-fits-all behemoth. Every feature of the framework is modular and highly customizable — so you'll never outgrow Directus.

Custom Workflows

Directus has limitless content authoring workflows. Create as many custom statuses as you'd like: Draft, Under-Review, Published, Soft-Delete. You can even set user permissions based on item status.

Learn More: Status Mapping

Form Layout

Our grid layout system allows you to order, group, and position fields, creating tailored forms for all your data collections. And you get over 50 different field interfaces out-of-the-box to choose from.

Learn More: Field Setup

Digital Asset Management

All of your files organized in one place. Directus supports: nested folders, drag-and-drop, IPTC metadata, multiple storage adapters, remote embed linking, and dynamic thumbnailing.

Learn More: Thumbnailer

Database Mirroring

Directus manages your content as pure SQL, with all system data stored separately. You have full control over the schema, including database optimizations, so you can even manage existing databases.

Learn More: Database Mirroring

Multilingual Support

Whether your project uses one language or 30 — Directus let's you manage multilingual content in as many locales as needed. Even the Directus App itself supports over 10 languages out-of-the-box.

Learn More: Internationalization
